AI-Powered Recommendation System for Food Delivery Platform

Muhammad Zubair Hasan
Jr Data Scientist, Kona Software Labs Ltd Apr 2021 - Nov 2022
Project Details Company

AI-Powered Recommendation System integrating RFM analysis and demographic features to predict user engagement with promotional offers in food delivery platform.

Project Overview

As a Jr Data Scientist at Kona Software Labs Ltd, I developed comprehensive recommendation systems for a food delivery platform that significantly improved user engagement and business metrics. The project involved building multiple AI models to predict user behavior and optimize promotional campaigns.

Key Components

1. Coupon Recommendation System

  • Built recommendation system integrating RFM (Recency, Frequency, Monetary) analysis
  • Incorporated demographic features for personalized recommendations
  • Increased coupon redemption rates by 18%
  • Analyzed 10M+ user data points using Cloudera

2. Customer Segmentation & Analysis

  • Developed KNN-driven customer segmentation reports
  • Identified top-performing coupons in each customer segment
  • Guided targeted marketing campaigns
  • Improved overall coupon ROI by 15%

3. Food Menu Recommendation Models

  • Implemented content-based recommendation models
  • Developed SVD-based collaborative filtering
  • Built hybrid recommendation models for similar food menu suggestions
  • Enhanced user experience through personalized menu recommendations

4. System Development & Deployment

  • Developed Flask APIs for recommendation services
  • Automated deployment pipeline with Bash scripting
  • Implemented CI/CD processes
  • Reduced deployment time by 50%

Impact & Results

  • 18% increase in coupon redemption rates
  • 15% improvement in overall coupon ROI
  • 50% reduction in deployment time through automation
  • 10M+ user data analyzed using Cloudera
  • Enhanced user engagement through personalized recommendations
  • Scalable architecture supporting high-volume data processing
