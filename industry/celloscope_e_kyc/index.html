Automated E-KYC Pipeline with NIST-compliant facial recognition, video liveness detection, and National ID extraction services.

Project Overview

As a Data Scientist at Celloscope Ltd, I developed a comprehensive automated e-KYC (Electronic Know Your Customer) pipeline that revolutionized the identity verification process. The system integrates multiple AI-powered components to provide a seamless and secure user verification experience.

Key Components

1. Face Recognition & Matching

  • Developed NIST-compliant facial recognition solution
  • Implemented advanced face matching algorithms for identity verification
  • Ensured high accuracy and low false positive rates

2. Video Liveness Detection

  • Built sophisticated video-based liveness detection system
  • Prevented spoofing attacks using deep learning models
  • Real-time processing with high accuracy

3. National ID (NID) Information Extraction

  • Automated extraction of information from National ID documents
  • OCR and computer vision techniques for data extraction
  • Validation and verification of extracted information

4. System Architecture & Deployment

  • Developed APIs using FastAPI framework
  • Containerized services using Docker
  • Implemented CI/CD pipeline for automated deployment
  • Improved service RPS (Requests Per Second) by 36% using model quantization

Impact & Results

  • 80% reduction in manual verification time
  • 500,000+ users currently served by the system
  • NIST compliance for facial recognition accuracy
  • 36% improvement in service RPS through model optimization
  • Scalable architecture supporting high-volume processing

Technologies & Skills

  • Computer Vision: OpenCV, TensorFlow, PyTorch
  • Face Recognition: Deep learning models, NIST compliance
  • Backend Development: FastAPI, Python
  • DevOps: Docker, CI/CD, Bash scripting
  • Model Optimization: Quantization, performance tuning
  • Document Processing: OCR, information extraction
