App-based Bangla Book Reader System for The Blind

Muhammad Zubair Hasan
Undergraduate Capstone Project Nov 2019 - Jan 2021
Project Details Code Demo

Accessibility Solution for visually impaired users to read Bangla books through image-to-text conversion and text-to-speech synthesis.

Project Overview

This capstone project addresses the accessibility challenges faced by visually impaired individuals in Bangladesh who want to read Bangla books. The system combines computer vision, natural language processing, and hardware components to create a comprehensive solution for converting printed Bangla text into audible speech.

System Architecture

1. Android Mobile Application

  • User-friendly interface designed for visually impaired users
  • Camera integration for capturing book pages
  • Image preprocessing and optimization
  • Communication with backend server for processing
  • Audio playback of converted text

2. Backend Server

  • Image processing and text extraction pipeline
  • Bangla OCR (Optical Character Recognition) system
  • Text-to-speech conversion using Google's Bangla TTS API
  • Audio file generation and delivery
  • Error handling and user feedback

3. Hardware Components

  • Book Stand: Stable platform for book positioning
  • Smartphone Mount: Secure camera positioning
  • Control Buttons: Easy navigation for visually impaired users
  • Arduino Microcontroller: Button input processing
  • Bluetooth Module: Wireless communication with smartphone

Technical Implementation

1. Bangla OCR System

  • Dataset Creation: Built custom Bangla word dataset (no open-source data available)
  • Word Segmentation: Used OpenCV's EAST detector for accurate word boundary detection
  • Model Training: EfficientNetB3 architecture for word classification
  • Accuracy: High precision in Bangla text recognition

2. Text-to-Speech Integration

  • API Integration: Google's Bangla Text-to-Speech API
  • Natural Speech: High-quality Bangla pronunciation
  • Speed Control: Adjustable playback speed for user preference
  • Audio Format: Optimized audio files for mobile playback

3. Hardware Integration

  • Arduino Programming: Button input processing and Bluetooth communication
  • Bluetooth Protocol: Reliable wireless data transmission
  • Power Management: Efficient battery usage for portable operation
  • Ergonomic Design: User-friendly interface for visually impaired users

System Workflow

  1. Page Capture: User positions book on stand and captures image using smartphone camera
  2. Image Processing: Backend server receives image and applies preprocessing techniques
  3. Text Extraction: EAST detector segments words, EfficientNetB3 classifies Bangla text
  4. Text Processing: Extracted text is cleaned and formatted for TTS conversion
  5. Speech Synthesis: Google's Bangla TTS API converts text to natural speech
  6. Audio Delivery: Generated audio file is sent back to mobile app
  7. Playback: User listens to the converted speech through smartphone speakers

Challenges & Solutions

Technical Challenges

  • Bangla Dataset: No open-source Bangla OCR dataset available
  • Solution: Created custom dataset with diverse Bangla text samples
  • Text Recognition: Complex Bangla script with multiple character forms
  • Solution: Used state-of-the-art EfficientNetB3 for robust recognition
  • Hardware Integration: Coordinating multiple hardware components
  • Solution: Modular design with clear communication protocols

Accessibility Challenges

  • User Interface: Designing for visually impaired users
  • Solution: Audio feedback and simple button controls
  • Book Positioning: Accurate page capture without visual feedback
  • Solution: Physical book stand with guides and audio cues

Impact & Results

  • Accessibility: Enables visually impaired users to read Bangla books independently
  • Innovation: First comprehensive Bangla book reader system for the blind
  • Technical Achievement: Custom Bangla OCR system with high accuracy
  • Hardware Integration: Complete hardware-software solution
  • User Experience: Intuitive interface designed for target users
  • Scalability: Framework can be extended to other languages

Technologies & Skills

  • Mobile Development: Android, Java, Camera API
  • Computer Vision: OpenCV, EAST detector, Image processing
  • Machine Learning: EfficientNetB3, Custom dataset creation
  • Natural Language Processing: Text preprocessing, TTS integration
  • Hardware Programming: Arduino, Bluetooth communication
  • Backend Development: Server-side processing, API development
  • Accessibility Design: User interface for visually impaired
