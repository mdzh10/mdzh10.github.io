2D Pose Detection from a Video

Muhammad Zubair Hasan
Personal Project December 2021
Code Demo

Computer Vision Project for 2D human pose detection from multiple videos using mmpose and mmdet APIs with Python multi-threading for concurrent processing.

Project Overview

This computer vision project focuses on detecting 2D human poses from multiple video sources using state-of-the-art pose detection APIs. The project implements Python multi-threading to process multiple videos concurrently, significantly improving processing efficiency.

Technical Implementation

1. Pose Detection Pipeline

  • mmpose API: Advanced pose estimation using OpenMMLab's mmpose framework
  • mmdet API: Object detection for human localization in video frames
  • 2D Pose Estimation: Accurate detection of human keypoints and joints
  • Real-time Processing: Efficient frame-by-frame pose analysis

2. Multi-threading Architecture

  • Concurrent Processing: Simultaneous processing of 4 video streams
  • Thread Management: Efficient thread allocation and synchronization
  • Frame Extraction: Parallel frame extraction from multiple videos
  • Resource Optimization: Optimal CPU utilization and memory management

3. Video Processing Workflow

  • Input Handling: Support for multiple video formats and sources
  • Frame Processing: Individual frame analysis with pose detection
  • Data Integration: Combining results from multiple video streams
  • Output Generation: Final video with pose detection overlays

Key Features

1. Multi-Video Processing

  • Parallel Processing: Handle multiple video inputs simultaneously
  • Scalable Architecture: Easily extendable to more video sources
  • Efficient Resource Usage: Optimized CPU and memory utilization
  • Error Handling: Robust error management for individual video streams

2. Pose Detection Accuracy

  • High Precision: Accurate detection of human keypoints
  • Robust Detection: Reliable pose estimation across various conditions
  • Real-time Performance: Fast processing for live applications

3. Output Generation

  • Visual Overlays: Pose detection visualization on video frames
  • Combined Output: Merged results from multiple video sources
  • Format Flexibility: Support for various output video formats

Performance & Results

  • Output Quality: High-quality final video with pose overlays
