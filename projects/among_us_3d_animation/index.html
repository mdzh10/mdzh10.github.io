"Among Us" 3D Animation Using Blender

Muhammad Zubair Hasan
Creative Project Aug 2020 - Nov 2020
Project Details Code Demo

3D Animation Project using Blender software to create animated content inspired by the popular "Among Us" game with custom character designs and scenes.

Project Overview

This creative project involved creating 3D animations inspired by the popular "Among Us" game using Blender software. The project showcases skills in 3D modeling, character design, animation, and creative storytelling through visual media.

Project Features

1. Character Design

  • Custom Among Us Characters: Unique character designs inspired by the game
  • 3D Modeling: Detailed character modeling in Blender
  • Texturing and Materials: Realistic textures and material properties
  • Rigging: Character skeleton setup for animation

2. Scene Creation

  • Environment Design: Custom 3D environments and settings
  • Lighting Setup: Professional lighting and atmosphere
  • Camera Work: Dynamic camera movements and angles
  • Composition: Artistic scene composition and framing

3. Animation

  • Character Animation: Smooth character movements and actions
  • Storytelling: Narrative-driven animation sequences
  • Timing and Pacing: Professional animation timing
  • Special Effects: Visual effects and particle systems

Technical Implementation

1. Blender Workflow

  • 3D Modeling: Polygon modeling and sculpting techniques
  • UV Mapping: Texture coordinate mapping for materials
  • Rigging: Armature creation and weight painting
  • Animation: Keyframe animation and motion capture

2. Rendering

  • Render Engine: Cycles or Eevee rendering engine
  • Quality Settings: High-quality render settings
  • Post-processing: Color grading and effects
  • Output Formats: Video and image export options

Technologies & Skills

  • 3D Software: Blender, 3D modeling, Animation
  • Character Design: Concept art, Character development
  • Animation: Keyframe animation, Motion design
  • Rendering: Render engines, Post-processing
  • Creative Skills: Storytelling, Visual design
  • Video Production: Video editing, Export optimization
