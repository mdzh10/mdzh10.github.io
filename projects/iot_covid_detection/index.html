IoT Based Automated Entry System for Covid-19 Symptom Detection

Muhammad Zubair Hasan
Personal Project Nov 2021 - Dec 2021
Project Details Code Demo

Multi-Sensor IoT System for automated Covid-19 symptom detection with three levels of risk assessment using Raspberry Pi and Firebase database.

Project Overview

This IoT-based system provides automated entry control with comprehensive Covid-19 symptom detection. The system integrates multiple sensors and computer vision to create a three-level risk assessment system for entry validation during the pandemic.

System Components

1. Entry & Authentication System

  • RFID RC522 Sensor: Contactless entry and login mechanics
  • User Identification: Automated user recognition and authentication
  • Access Control: Entry permission based on health assessment

2. Mask Detection System

  • Camera Integration: Real-time video feed for face analysis
  • ResNet-50 Model: Pre-trained model fine-tuned on facial mask dataset
  • 99% Accuracy: High precision mask detection
  • Real-time Processing: Instant mask detection results

3. Health Monitoring Sensors

  • MLX90614 Temperature Sensor: Non-contact body temperature measurement
  • MAX30100 SPO2 Sensor: Blood oxygen saturation monitoring
  • Vital Signs Analysis: Comprehensive health parameter assessment

4. Data Management

  • Firebase Database: Cloud-based data storage and management
  • User Records: Comprehensive health and access logs
  • Real-time Updates: Instant data synchronization
