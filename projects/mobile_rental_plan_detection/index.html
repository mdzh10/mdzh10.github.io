Muhammad Zubair Hasan
Hackathon Project September 2021
Code Data

Multi-class Classification for mobile rental plan detection using LightGBM and XGBoost with advanced feature engineering and model stacking techniques.

Project Overview

This hackathon project focused on developing a machine learning solution for mobile rental plan detection. The challenge involved multi-class classification using ensemble methods and advanced feature engineering techniques to improve prediction accuracy.

1. Exploratory Data Analysis (EDA)

  • Feature Distribution: Statistical analysis of all input variables
  • Correlation Analysis: Identification of feature relationships
  • Data Quality Assessment: Missing value and outlier detection

2. Feature Engineering

  • New Feature Generation: Created derived features from existing data
  • Feature Transformation: Applied appropriate scaling and encoding
  • Feature Selection: Identified most predictive features
  • Domain Knowledge Integration: Incorporated business insights

3. Class Imbalance Handling

  • Imbalance Detection: Analyzed class distribution in target variable

Machine Learning Pipeline

  • Model Comparison: Evaluated performance across different algorithms
